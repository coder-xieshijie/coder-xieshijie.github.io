    2. 购买套餐
    3. 安装clashx
    4. 配置上网
  2. 接码平台：sms
    1. 先注册登录
    2. 点击右上角的余额，下拉点击充值
    3. 选择支付宝账号充值，PS：这个最低充值2美元，最开始没有最低额度的，因为chatgpt爆火，大量国人涌入，导致平台抬高了充值的价格，而且支付金额低于10美元时，佣金为3% + 0.03美元,要是有其他充值方式也可以不用选择这个
    4. 我充值了10美元，付款之后直接到账
    5. 点击左侧的服务列表中的openai服务，选中
  3. 注册google账号
  4. 注册chatgpt，成功！

注意事项

  • 科学上网的节点不要选择香港，最好选美国或者日本

科学上网

科学上网平台还是建议选择大一点的平台，买贵一点的套餐，太便宜的那种机场ip被大量人使用，openai就会直接把这个ip拉黑，即便注册成功了后续登录也会被限制

我使用的平台sockboom, 点击这里跳转到sockbook平台

定期有一些活动，每天登录一次可以免费领流量

节点列表，这个各个平台基本差不多，都是一些常见的

  1. 美国
  2. 俄罗斯
  3. 日本
  4. 香港
  5. 新加坡
  6. 土耳其
  7. 韩国
  8. 马来西亚
  9. 等等

套餐列表

PS:我开了VIP(可以多选一些节点)，买了一年的套餐，一共花了￥80

使用非常简单，按照平台的提示即可

我使用的是mac，点击到mac选项，先下载clashx并安装，

然后点击一键导入clashx配置，更新完成之后，点击添加

选择美国/日本的接口(需要注意，选节点不要选香港，香港已经被openai划入黑名单了)，并设置全局代理即可科学上网

输入https://tool.lu/ip/，验证一下当前的ip

接码平台

https://sms-activate.org/

使用：

  1. 先注册登录
  2. 点击右上角的余额，下拉点击充值
  3. 选择支付宝账号充值
    1. PS：这个最低充值2美元，最开始没有最低额度的，因为chatgpt爆火，大量国人涌入，导致平台抬高了充值的价格，而且支付金额低于10美元时，佣金为3% + 0.03美元,要是有其他充值方式也可以不用选择这个
  4. 我充值了10美元，付款之后直接到账
  5. 点击左侧的服务列表中的openai服务，选中

选择openai服务

看一下价格，选一个价格低的。

一个国家有两个价格，前一个是批发价，后一个是零售价，选后面数字低的

目前直接使用邮箱的方式基本已经走不通了

提示

  • Signup is temporarily unavailable, please check back in an hour.

注册Google账号

目前依然有效的方式，Google账号

打开一个新的无痕窗口

访问：https://accounts.google.com/， 点击这里直达

语言选择English

点击创建个人账号

填写自己的账号密码

确认你的手机号

去刚刚注册的sms接码平台购买一个最便宜的Google服务虚拟号

我选的是印度尼西亚

把接码平台的号码粘帖进去并点击next

如果出现：This phone number cannot be used for verification.

去接码平台在重新选一个，先把当前激活的×掉，这样刚刚的费用就会推给你(如果20分钟之内没收到短信也会退给你，还是蛮良心的)

我又重新选了一个巴西的账号，就成功验证了

验证码通过之后，填写自己的相关信息，生日和性别

skip

aggre政策

google账号申请完成！

注册ChatGPT

重新登录https://chat.openai.com/

选择continue as google

选择你刚刚注册的google邮箱

成功进入手机号验证环节！

再次到接码平台，选择openai服务，购买一个虚拟账号

填写手机号并点击NO，然后点击send code via sms

填写验证码

chatgpt申请成功！
